Peupasswd Advanced Password Security Software

Peupasswd is an advanced password security software written in python use in generating unique password for each of each service i.e you can generate special password for each of your social media or services with just one master password; see also neupasswd

How It Works

Peupasswd takes in a master password, the service name you wanna use the password for and the length of the password u want(never worry about the length you can choose as much as possible you dont need to cram it). Then it encrypt the master password in a way you can nerver guess of. Also you can retrieve/check ur passwordagain by doing same way.

Instructions:-

  • Go here.
  • Create a master password in your brain.
  • Enter the master password.
  • Enter the service name note: you will need this when trying to check/retrieve password.
  • Enter the length you want ur password to be.
  • Now click generate to generate the unique password. you can always generate the same password later by simply giving the same master password and service name. but no one could generate the same password as long as you dont share your master password to anyone. this is why I call it Advanced Password Security Software.
